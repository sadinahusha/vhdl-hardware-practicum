

# VHDL Hardware Practicum

This repository contains my VHDL assignments, simulations, and documentation completed during my hardware practicum.

## Technologies

* VHDL
* GHDL
* GTKWave

## Assignments

### Assignment 01

Introduction to the VHDL workflow, simulation, and waveform analysis.

### Assignment 02

Logic gates, signal assignments, and transport delay simulations.

### Assignment 03

Implementation of arithmetic circuits including Full Adders, Ripple Carry Adders, and Carry Lookahead Adders.

### Assignment 04

Design and simulation of multiplexers, demultiplexers, and encoders.

### Assignment 05

Implementation and testing of SR Latches, D Latches, and JK Flip-Flops.

## Learning Outcomes

* Digital Logic Design
* VHDL Development
* Testbench Creation
* Simulation and Verification
* Sequential and Combinational Circuits
